Install Banner Should be Triggered

This page meets the criteria of showing Web App install banner in Chrome for Android:

  • A Service Worker registered.
  • A Web App manifest file with short_name and 144x144 png icon defined.
  • The icon is image/png.
  • The site is served over HTTPS.
Chrome applies user engagement detection to prompt install banner, this can be disabled in chrome://flags to always show this banner.
To show Google Play install banner, set related_applications
